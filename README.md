# What do computer scientists do and why does it matter?  
Computer scientists study the theoretical foundations of information and computation in order to design algorithms, software and hardware to solve complex, real-world problems.  
This matters as it helps to improve technological efficiency and innovation, driving advancements in medicine, business, and daily life.
